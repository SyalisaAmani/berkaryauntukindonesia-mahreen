# berkaryauntukindonesia-mahreen
Landing page for Mahreen Indonesia. Built to empower young talents, bridge the digital gap, and transform creative ideas into impactful digital solutions.  Tech Stack Frontend: HTML5, CSS3, &amp; JavaScript (ES6+).  Styling: Tailwind CSS. Intersection Observer API &amp; AOS.  Deployment: Netlify.  Icons/Assets: Heroicons &amp; Unsplash API.
